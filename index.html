Welcome to the Behavioural Questions RESTful API!

Records will be in json format:

{
"_id": a unique ID Number, in BSON format allocated automatically.
"question": a String - "Question must be in the format - Can you give me an example of a time when X? Don't be tempted to lead",
"answer": a String - "Answer with CAR - Cause, Action, Result. Don't use hypotheticals, don't generalise - think of a specific example",
"answered": a boolean "TRUE",
"values": An array of strings with leadership values [],
}

Accessing the API

Return a Random Question

https://behavioural-recruitment.thewoodenman.repl.co/random

Return a Question by ObjectId

Send a GET request to https://behavioural-recruitment.thewoodenman.repl.co/api/
appending the URI with /id/ObjectId
Where Number is your unique Object Id referring to the question you want
Object Ids are in the standard BSON format.

Submit a Question

Send a POST request to https://behavioural-recruitment.thewoodenman.repl.co/api/
Append the URI with /addquestion omitting the ObjectId as these will be allocated automatically,
Include the new question in JSON format in the body in correct format above.

Replace a Question

Send a POST request to https://behavioural-recruitment.thewoodenman.repl.co/api/
Append the URI with /api/byId/:idToReplace substituting idToReplace with the ObjectId of the target question.
submit your replacement question in JSON format as the body of a POST request
