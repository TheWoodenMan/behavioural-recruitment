Welcome to the Behavioural Questions RESTful API Guide

Records will be in the following JSON format:

{
"_id": a unique ID Number, in BSON format allocated automatically.
"question": a String - "Question must be in the format - Can you give me an example of a time when X? Don't be tempted to lead",
"answer": a String - "Answer with CAR - Cause, Action, Result. Don't use hypotheticals, don't generalise - think of a specific example",
"answered": a boolean "TRUE",
"values": An array of strings with leadership values [],
}

    Accessing the API

    Return a Random Question

    https://behavioural-recruitment.thewoodenman.repl.co/questions/random/pickone

    Return a Question by ObjectId

    Send a GET request to https://behavioural-recruitment.thewoodenman.repl.co/questions/{id}
    replacing {id} in the URI with the ObjectId of your target question
    Object Ids are in the standard BSON format.

    Return a Question by a Single Value

    Send a GET request to https://behavioural-recruitment.thewoodenman.repl.co/questions/values/{value}
    replacing {value} in the URI with your target leadership value in string format
    Object Ids are in the standard BSON format.

    Submit a Question

    Send a POST request to https://behavioural-recruitment.thewoodenman.repl.co/questions/addone
    Omit the ObjectId in the JSON as these will be allocated automatically,
    Include the new question in JSON format correctly in the format displayed above.

    Replace a Question

    Send a POST request to https://behavioural-recruitment.thewoodenman.repl.co/questions/{id}/replace
    replacing {id} in the URI with the ObjectId of your target question
    submit your replacement question in JSON format as the body of the POST request

    Replace the values of a Question

    Send a POST request to https://behavioural-recruitment.thewoodenman.repl.co/questions/{id}/values/replace
    replacing {id} in the URI with the ObjectId of your target question
    submit your replacement values in JSON format, but as a series of strings within an array. as the body of the POST request
    { "values": ["optimistic", "results oriented", "sense of humour", "speed", "versatility"] }
